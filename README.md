# user-card
a web component using Polymer library 3.

### Overview

![](./user-card.png)

### Installation

1. use `npm install` to install node dependencies:

   ```shell
   $ npm install
   ```

2. Start the component, here I use `browser-sync` to proxy `polymer-cli` port:

   ```shell
   $ npm start
   ```
